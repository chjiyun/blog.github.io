windows命令行利器-Cmder

发表于 2017-11-15 | 分类于 教程

Cmder 是一个软件包，它被创建出来替代没有漂亮的控制台模拟器的Windows上。它也是一个增强型的命令行工具，不仅可以使用cmd下的所有命令，更厉害的是还可以使用shell命令和Git命令。
